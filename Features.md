# 🌟 Features of My-4-Blocks ✨

## 🏰 The Sanctuary (Admin Dashboard)
- **Role-Based Bastion**: Password-protected access at `/v0/admin` for system administrators.
- **GEPA Report Vault**: Real-time browsing and downloading of GEPA analysis artifacts (baseline, evolved, metrics).
- **Knowledge Graph Tapestry**: Interactive iframe-based visualization of emotional connections and knowledge nodes.
- **Run Chronology**: Automated grouping and sorting of analysis runs from the filesystem.

## 📱 The Native Experience
- **iOS & Android Mastery**: Flutter-based native application (`/mobile/`) for therapeutic chat on the go.
- **SSE Stream Alchemy**: Real-time Server-Sent Events extraction for smooth, streaming AI responses.
- **Fluid Motion**: High-fidelity animations using `flutter_animate` for a serene UX.

## 🧠 The Intelligence Engine
- **Unified RAG Library**: Shared knowledge base across all variants (331 artisanal chunks).
- **Hybrid Search Alchemy**: 70% semantic + 30% keyword-boosted retrieval.
- **Local Sentiment Wisdom**: Offline intensity detection for nuanced emotional understanding.
- **Dr. Parr's Formula**: Responses strictly grounded in the "Four Blocks" framework (Anger, Anxiety, Depression, Guilt).

## 🎨 The Digital Canvas (Web Variants)
- **V0 Mastery**: High-performance, SEO-optimized Next.js implementation with Shadcn/UI.
- **Ethereal Glassmorphism**: SERP-ready, beautiful interfaces with DM Sans and Cormorant Garamond typography.
- **Voice Mode**: 9 artisanal voices and multiple conversation styles (Direct, Casual, Warm, Professional).
