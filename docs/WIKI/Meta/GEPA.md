# 🧬 GEPA Optimizer ✨

**GEPA (Genetic-Pareto Prompt Evolution)** is the optimization engine used in the **My 4 Blocks** project to evolve agent prompts and skills. It is powered by the [**DSPy**](./DSPy.md) framework.

---

## 🌟 **The Evolutionary Mechanism**

GEPA treats prompts as "genetic material" that can be mutated and selected based on fitness:

1.  **Seed**: Starts with a baseline prompt or skill.
2.  **Mutate**: Generates variants of the prompt using a teacher model (e.g., GPT-5.5).
3.  **Evaluate**: Tests each variant against a set of "Golden Examples" (high-quality input/output pairs).
4.  **Select**: Uses a **Pareto Frontier** approach to find variants that balance multiple metrics (e.g., accuracy, brevity, brand voice).

---

## 🔮 **In the My 4 Blocks Ecosystem**

GEPA is used to ensure that the [**Hermes Agent**](./Hermes_Agent.md) provides responses that are:
- **Clinically Accurate**: Aligned with [**Dr. Parr's**](../People/Vincent_Parr.md) formulas.
- **Compassionate**: Using the "Spellbinding" brand voice.
- **Efficient**: Minimizing token usage while maximizing insight.
